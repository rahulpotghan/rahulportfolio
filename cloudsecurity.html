Security Application- Log Management

About the project

Log Manager provides a simple, yet powerful log collection solution for customers concerned with IT security and compliance regulations. Different deployment types are available for Log Manager, however, existing workflow has gaps which are addressed and new service design was proposed.

My role

I was responsible for user flows, wireframes, functional specifications, and overall product design. I also collaborated with the analytics team to add features like one tap driver's travel plan, in-app marketplace etc.

Service Blueprint

A Service Blueprint can clearly identify the interactions between the user, touchpoints, and service employees, including the activities that the user can directly see and those that user does not see. Therefore, it is a very powerful tool that can be used to better deliver a successful customer experience holistically; it is a great tool in the way that it makes it easy to explain a long-running and complicated process in just a few minutes, which adds tremendous value in getting multiple minds together to collaborate on a problem.
The Service Blueprint for the Log Management Setup Wizard starts off with the user signing up for the service, setting up the sources and going into the product, stepping through the wizard, capturing user action and emotions. The existing workflow is marked in green. Existing workflow is mapped and then customer emotions are identified in yellow boxes. Wherever the emotion is not entirely positive, there is a red circle next to it identifying them as pain points in the journey.
